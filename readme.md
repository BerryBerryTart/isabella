# Isabella

Created with Django and React.js

### Requirements
- Python3
- Pip3
- Node.js

### Installation
```sh
python3 -m venv env
env/bin/activate # On Windows use `env\Scripts\activate`
git clone {this project}
cd app
pip install -r requirements.txt
npm install
npm run dev
python3 manage.py migrate
python3 manage.py runserver
```
